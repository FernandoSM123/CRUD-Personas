Registro de Personas

Aplicación web que permite realizar un CRUD(Create,Read,Update,Delete), sobre un registro de personas con base en 3 datos: Nombre,Apellido,Edad. Dicha aplicación almacena la información en la memoria local del navegador, tambien permite realizar la acción de limpiar completamente el registro
